# Customer Churn Prediction

## 📌 Overview
This project predicts whether a customer is likely to churn using Machine Learning techniques.

## 🚀 Problem Statement
Customer churn is a major challenge for businesses. The goal of this project is to identify customers who are likely to leave so that proactive actions can be taken.

## 🧠 Approach
- Data preprocessing (handling missing values, encoding categorical variables)
- Feature selection
- Logistic Regression model
- Handling class imbalance using weighted training

## 📊 Results
- Achieved ~73% accuracy
- Improved recall for churn prediction (important for business impact)

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn

## 📈 Key Learnings
- Importance of data preprocessing
- Handling imbalanced datasets
- Understanding precision vs recall trade-off

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the project:
   python src/main.py
